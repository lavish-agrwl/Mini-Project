# Quick Start Guide

## One-Command Setup (After MongoDB is running)

```bash
# Install all dependencies
npm run install-all

# Seed demo data (optional)
npm run seed

# Then run these in separate terminals:
# Terminal 1:
npm run server

# Terminal 2:
npm run client
```
